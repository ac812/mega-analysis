doRelated - Multi-purpose script with the following commands:
    Takes a plink type of file as standard input
    (i.e. individualId FamilyId Population) - tab separated

    split           - Generates plink files per population
    IBD             - Calculates IBD
    statIBD         - Gathers IBD statistics
    removeCloserIBD - Removes inbreed individuals (requires threshold)
    removeDistant   - Removes distant individuals (probably not from the pop)
